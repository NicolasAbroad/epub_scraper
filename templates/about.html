WNScraper

Description

WNScraper is a simple web scraper designed to get the content of any web novel from the website https://ncode.syosetu.com/, and turn it into an epub file.

Note: epub files are generated per volume. If there are no volumes, one epub file containing every chapter is generated.

Reason for making this

It's a fun way to teach myself how to program.
